## Quick Start
```sh
# Initializing Virtual Env
python3 -m venv .venv \
    && source .venv/bin/activate \
    && pip install -r requirements.txt

# Kafka Producer
python producer.py

# In another terminal execute "kafka consumer"
# Don't forget to activate python venv before running ;)
python consumer.py
```
